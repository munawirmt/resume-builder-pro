# Resume Builder Pro 📄✨

Create professional, ATS-friendly resumes in minutes. 100% Offline. No Ads. No Login.

![App Screenshot](screenshots/home.png)
![App Screenshot](screenshots/template.png)
![App Screenshot](screenshots/pdf.png)

## 🌟 Features

- **15+ Professional Templates** - Modern, Classic, Creative & Minimal designs
- **ATS Optimized** - Get past Applicant Tracking Systems easily  
- **PDF Export** - One-tap A4 PDF export with no watermark
- **Live Preview** - See your resume update in real-time
- **All Sections** - Profile, Education, Experience, Skills, Projects, Certificates
- **Save & Load** - Import/Export your resume data as JSON
- **100% Offline** - Works without internet connection
- **Lightweight** - Under 10MB

## 🚀 Getting Started

### Download APK
[Download Latest APK](https://resume-builder-pro-c58t.onrender.com)

### Build from Source
1. Clone this repository
2. Open in Kodular/App Inventor
3. Import assets from `/assets` folder
4. Export as APK
